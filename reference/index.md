# Package index

## Select, browse, and filter

The package’s single entry point – browse tissues/systems, pull whole
organ systems, or select specific organs (optionally attaching values).

- [`anatomogram_select()`](https://petertanksley.github.io/anatomogramdata/reference/anatomogram_select.md)
  : Browse, select, or filter anatomogram tissues – by organ, by organ
  system

## Data

The bundled tissue polygon data and the hand-curated organ-system lookup
it’s joined against.

- [`hgMale`](https://petertanksley.github.io/anatomogramdata/reference/hgMale.md)
  : Human male tissue polygon data
- [`hgFemale`](https://petertanksley.github.io/anatomogramdata/reference/hgFemale.md)
  : Human female tissue polygon data
- [`organ_systems`](https://petertanksley.github.io/anatomogramdata/reference/organ_systems.md)
  : Organ system membership for anatomogram tissues
